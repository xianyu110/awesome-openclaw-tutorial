```markdown
### 2.2.1 验证Docker镜像

在拉取镜像前，先验证镜像是否存在：

```bash
# 搜索OpenClaw镜像
docker search openclaw

# 查看可用标签
curl -s https://hub.docker.com/v2/repositories/openclaw/openclaw/tags/ | grep name
```

**如果镜像不存在或拉取失败**：

1. **配置镜像加速**（推荐）
   ```bash
   # 编辑Docker配置
   sudo nano /etc/docker/daemon.json
   
   # 添加国内镜像源
   {
     "registry-mirrors": [
       "https://docker.mirrors.ustc.edu.cn",
       "https://hub-mirror.c.163.com"
     ]
   }
   
   # 重启Docker
   sudo systemctl restart docker
   ```

2. **使用云端部署**（最简单）
   - 腾讯云Lighthouse：20元/月
   - 火山引擎：9.9元/月
   - 百度云O计划：0.01元/首月
   
   详见：[云端一键部署](#云端一键部署)

3. **使用本地安装**（Mac用户）
   ```bash
   curl -fsSL https://openclaw.ai/install.sh | bash
   ```

### 2.2.2 故障排查

**403错误**：
- 原因：Docker Hub访问限制或镜像不存在
- 解决：配置镜像加速或使用云端部署

**网络超时**：
- 原因：网络连接问题
- 解决：检查网络或使用代理

**权限错误**：
- 原因：Docker权限不足
- 解决：`sudo usermod -aG docker $USER`
```

# 🔍 搜索功能使用指南

> 本教程的搜索功能专注于标题搜索，帮助你快速找到需要的章节

---

## 📖 搜索说明

### 搜索范围

本站搜索功能**仅搜索文档标题**，不搜索正文内容。这样设计的原因：

- ✅ **更精准**：直接定位到章节标题
- ✅ **更快速**：搜索速度更快
- ✅ **更清晰**：结果更容易理解

### 支持的内容

搜索覆盖以下内容的标题：

- 📚 **15章正文**：第1-15章的所有标题
- 📖 **8个附录**：附录A-H的标题
- 💡 **示例文档**：examples目录下的文档标题
- 📄 **其他页面**：README、学习路径等

## 🎯 搜索技巧

### 1. 使用关键词

**推荐做法**：
```text
✅ 飞书
✅ 配置
✅ Skills
✅ 问题
✅ API
```

**不推荐**：
```text
❌ 如何配置飞书Bot（太长）
❌ 我想知道怎么安装（口语化）
```

### 2. 中文搜索

支持中文关键词搜索：

```text
飞书 → 找到所有包含"飞书"的标题
配置 → 找到所有包含"配置"的标题
问题 → 找到所有包含"问题"的标题
```

### 3. 英文搜索

支持英文关键词搜索：

```text
API → 找到所有包含"API"的标题
Skills → 找到所有包含"Skills"的标题
Bot → 找到所有包含"Bot"的标题
```

### 4. 组合搜索

使用空格分隔多个关键词：

```text
飞书 配置 → 找到同时包含"飞书"和"配置"的标题
API 问题 → 找到同时包含"API"和"问题"的标题
```

### 5. 模糊匹配

搜索支持前缀匹配：

```text
配 → 可以匹配"配置"
问 → 可以匹配"问题"
Ski → 可以匹配"Skills"
```

## 🚀 高级功能

### 实时搜索

输入2个字符后，搜索会自动开始（延迟300ms），无需点击搜索按钮。

### 结果高亮

搜索结果中，匹配的关键词会用黄色背景高亮显示。

### 分类标签

搜索结果会显示内容分类：
- 📚 文档 - docs目录下的文档

## ❓ 常见问题

### Q1：为什么搜索不到某些内容？

**A**：搜索只匹配标题，不搜索正文内容。如果标题中没有你搜索的关键词，就不会出现在结果中。

**解决方案**：
- 尝试使用标题中可能出现的关键词
- 查看目录，找到相关章节
- 使用浏览器的页内搜索（Ctrl+F / Cmd+F）

### Q2：搜索结果太多怎么办？

**A**：使用更具体的关键词，或组合多个关键词。

**示例**：
```text
❌ 配置 → 结果太多
✅ 飞书 配置 → 结果更精准
```

### Q3：搜索结果太少或没有结果？

**A**：尝试以下方法：

1. **使用更短的关键词**
   ```
   ❌ 飞书Bot配置 → 可能没结果
   ✅ 飞书 → 有结果
   ```

2. **检查拼写**
   ```
   ❌ Skils → 拼写错误
   ✅ Skills → 正确
   ```

3. **使用同义词**
   ```
   问题 = 故障 = 错误
   配置 = 设置
   ```

### Q4：搜索速度慢？

**A**：首次加载时需要下载搜索索引（约几KB），之后搜索会很快。如果持续慢，请：

1. 刷新页面
2. 清除浏览器缓存
3. 检查网络连接
